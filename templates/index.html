    Fast Linux Deployment Toolkit

    FLDT (Fast Linux Deployment Toolkit) uses a variety of open source tools to quickly image and configure Linux hardware, and is versatile enough to handle either single machines or large batches of systems.

    It also provides methods for running post-image scripts, such as machine-specific configurations such as automatically setting hostnames and forcing password change on next boot.

    FLDT is used in three simple steps:

    Select an Image and Post-Image Action

    Select one of the prepared images and choose an action to take following the application of the image.

    Upload a *.csv To Identify the Machines

    FLDT uses a simple *.csv to match MAC addresses to Hostnames.

    Start the Batch Imaging Process

    Once preparations are complete and the devices are ready for imaging, simply click 'Start Multicast' to image a large batch of computers very quickly.
    We have had success with imaging up to 100 computers at once, completing in less than 20 minutes.
    Imaging is accomplished by using partclone to copy and restore only the used blocks from the file system.
    Images can be transferred to clients via NFS or multicasted via udpcast.
